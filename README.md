# Satyam Interface (JARVIS-style 3D Portfolio)

## Deploy on GitHub Pages
1. Upload all files to your GitHub repo (same structure).
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** / **root**
5. Save → open your site link.

## Edit your info
Open: `data/profile.json`

Update:
- email, phone
- GitHub link
- resume link
- skills, projects, experience

## Notes
- Voice greeting plays only after clicking **ENTER INTERFACE** (browser rule).
- Contact form uses `mailto:` (works on GitHub Pages).
